Prime the Lab Environment

Everything below keeps you from pausing mid-demo to install packages.

Devices

VS Code + Copilot

Verify you're signed into GitHub with access to Copilot and have the extension enabled.

Connectivity

Okta Org Ready

Toolchain

Node + Package Manager

Lock to Node LTS (18.17+) and confirm npm/pnpm versions to avoid compile surprises.

Core Tools

  • VS Code

    The command center. Ensure you are on the latest version.

    code --version

  • Node.js (LTS) & NPM

    Required for Next.js. Version 18.17.0 or later is strictly required.

    node -v && npm -v

Package Management

macOS / Linux

Homebrew is essential for managing dependencies.

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Windows

Recommend using Winget or Chocolatey.

winget install -e --id Microsoft.VisualStudioCode

VS Code Extensions

Install these to enable syntax highlighting and formatting for the Vibe Stack.

ESLint Linting
Prettier Formatting
Tailwind CSS IntelliSense

AI Coding Assistants

Required for Phase 1 (Canvas) and Phase 2 (VS Code). Phase 3 uses Copilot's native AI.

GitHub Copilot
